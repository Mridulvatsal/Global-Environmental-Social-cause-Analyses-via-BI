# 🌍 Global Economic Indicators Analysis

## 📌 Project Overview

This project presents a comprehensive analysis of global economic and environmental indicators including:

* GDP (Gross Domestic Product)
* Population Growth
* Energy Consumption
* CO₂ Emissions
* Freshwater Usage

The analysis is performed using **Microsoft Power BI**, enabling interactive visual exploration and data-driven insights.

---

## 🎯 Objectives

* Analyze GDP trends across countries (1990–2020)
* Study population growth dynamics
* Examine energy usage and carbon emissions
* Evaluate freshwater consumption patterns
* Build an interactive dashboard for global insights

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query (Data Cleaning)
* DAX (Data Analysis Expressions)
* Microsoft Excel

---

## 📊 Dashboard Features

* 📈 Multi-country GDP trend analysis
* 🌎 Interactive world map visualization
* 📊 KPI cards (GDP, Population, Energy, CO₂, Water)
* 🍩 Donut chart (Energy vs CO₂ distribution)
* 🎯 Slicers (Country, Year, Indicator)

---

## 📂 Dataset

* Source: World Bank Open Data (WDI)
* Time Range: 1990–2020
* Countries: ~195

---

## 🔍 Key Insights

* China shows exponential GDP growth (1990–2020)
* Developed countries show CO₂ reduction trends
* India & Africa show rapid population growth
* Water stress is critical in Middle East & South Asia
* Energy consumption strongly correlates with GDP
